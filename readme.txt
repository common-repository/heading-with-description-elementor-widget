== Installation ==

#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'Heading With Description Elementor Widget'
3. Install the plugin
3. Activate the 'Heading With Description Elementor Widget' plugin from the 'Plugins' menu in WordPress. 


#### Manually

1. Download the plugin https://downloads.wordpress.org/plugin/heading-with-description-elementor-widget.1.0.zip 
2. Unzip the zip file 
1. Upload the heading-with-description-elementor-widget/ folder to the /wp-content/plugins/ directory. 
2. Activate the 'Heading With Description Elementor Widget' plugin from the 'Plugins' menu in WordPress.
